# 日志检索工具

一个基于 Flask 的轻量级 Web 日志检索工具，支持通过浏览器对指定目录下的日志文件进行全文检索，并内置 AND / OR / NOT 逻辑运算支持。

---

## 功能特性

- **Web 可视化界面**：无需命令行，打开浏览器即可检索日志
- **逻辑运算符支持**：支持 `AND`、`OR`、`NOT` 组合检索，快速定位目标日志
- **多目录级检索**：可指定一个或多个日志目录，自动扫描所有目录下的文本文件
- **结果高亮展示**：匹配关键词自动高亮，显示文件来源与具体行号
- **响应式布局**：适配桌面端与移动端

## 快速开始

### 1. 安装依赖

确保已安装 Python 3，然后执行：

```bash
pip install -r requirements.txt
```

> 本项目仅依赖 `Flask`。

### 2. 启动服务

```bash
python app.py
```

服务默认运行在 `http://0.0.0.0:5000`。

### 3. 使用检索

打开浏览器访问 `http://localhost:5000`：

1. 在上方输入框填入**日志目录路径**（如 `D:\logs` 或 `/home/user/logs`）。可点击「+ 添加目录」增加多个目录，点击「删除」移除不需要的目录；所有非空目录都会参与检索
2. 在检索框输入关键词，支持以下逻辑语法：

| 语法示例 | 含义 |
|---------|------|
| `error AND timeout` | 同时包含 error 和 timeout |
| `error OR warning` | 包含 error 或 warning 之一 |
| `timeout NOT error` | 包含 timeout 但不包含 error |
| `client_msg AND 3910 NOT error` | 组合使用 |

3. 点击「检索」按钮或按回车键查看结果

---

## 检索语法说明

- 默认连接方式为 **AND**（即所有关键词都必须出现）
- 使用 `AND`、`OR`、`NOT` 明确指定逻辑关系（不区分大小写）
- 检索不区分大小写
- 匹配规则为**子串匹配**（关键词出现在行内任意位置即命中）

## 示例日志格式

本项目 `log_files/` 目录下附带了一份示例日志，单条格式如下：

```json
{"log":"2026-04-02T18:19:37.121 [FATAL] [user/user_func.js:250] send_response_data 1707543705 9795 {...} {...}"}
```

你可以将实际业务日志放入该目录，或在前端页面指定其他目录进行检索。

## 注意事项

- 服务启动后会读取本机指定目录下的文件内容，请确保运行用户拥有对应目录的读取权限
- 当前版本为逐行文本匹配，适用于大多数结构化 / 半结构化日志场景
- 建议在本地或内网环境使用，如需公网访问请配置反向代理并增加身份验证
